三.React中的语法注意

1.React中的key用法
vue和react都是在操作虚拟dom，并且根据diff算法进行新旧dom对比，从而更新dom
如果不使用key，Vue会使用一种最大限度减少动态元素并且尽可能的尝试修复/再利用相同类型元素的算法
使用key，它会基于key的变化重新排列元素顺序，并且会移除key不存在的元素
key值的存在保证了唯一性，可以用于dom的重新渲染或是就地复用

3.两种区别
使用class关键字创建的组件有自己的私有数据生命周期函数————有状态组件
使用function创建的组件只有props没有自己的私有数据和生命周期函数————无状态组件
无状态组件运行效率稍高
本质区别：有无state属性 和 有无生命周期函数

props和state/data 的区别：
props由外界传递 只读不能重新赋值
state/data都是组件私有 (通过Ajax获取的数据一般都是私有数据)
state/data可读可写
