5.在组件中使用css样式表
首先安装 cnpm i style-loader css-loader -D
然后在module的rules中配置

*可以打包处理css样式表的第三方 因此在引入css文件的时候能够处理

*在CmList 中 import cmlist from ‘@/css/cmlist.css’
*并在h1标签中引入 className=”title”
*但是此时样式表并不只在CmList中生效 反而 实在全局上生效

*React中没有类似的指令可以解决作用域的问题
*但是 可以利用模块化解决 本来导入import cmlist from ‘@/css/cmlist.css’时 cmlist是一个空对象 但是启用模块化之后 能接收到内容
**如何变成模块化
在css-loader后面利用?追加参数 modules这一个固定参数表示 为普通的css样式表启用模块化

[path] [name] [local] [hash:length]
