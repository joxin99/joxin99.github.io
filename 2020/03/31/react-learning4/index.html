七.React的组件生命周期

React的生命周期分为三个阶段
1.创建
一辈子执行一次
componentWillMount
render
componentDidMount
2.运行
按需 根据props属性或者state状态的改变 有选择性地执行0到多次
componentWillReceiveProps
shouldComponentUpdate
componentWillUpdate
render
componentDidUpdate
3.销毁
一辈子只执行一次
componentWillUnmount
