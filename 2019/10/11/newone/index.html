2.先前已经建立了博客 但是还没来得及写文章 荒废已久 完全记不起自己到哪一步了 几乎是从头再来过 研究了一会 便记录下这回的步骤吧

3.发布文章的步骤:
(1) 终端里 ls 查看所有文件 cd myblog 到目录下开始运行一系列指令
(2) hexo new ‘新建一篇文章’
(3) 在 source/_posts 目录下查看此文章 进行编辑
(4) 输入 hexo g 之后 输入 hexo s 可以在本地localhost:4000查看到文章

(5) 但是在joxin.github.io上还未能查看 因此需要将其推送到github上保存发布
(6) 输入 hexo clean 接着输入 hexo g -d 部署到站点上 便可以在博客上看到新发布的文章了
